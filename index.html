组件

创建一个组件
注意： React.Component{} 直接大括号{}，没有小括号()
render(){} 渲染函数(){} 都要加
return() 是返回标签，必加小括号()

组件的后缀可以是js，也可以是jsx
一个react项目是有成千上万个组件组成

props属性

组件的复用性很重要
// props子元素使用他不可以被修改，修改需要到父元素

state

组件中的状态state
以前我们操作页面元素的变化，都是修改dom，操作dom
但是有了react框架，我们不再推荐直接操作dom，页面元素的改变使用state进行处理

注意：箭头函数可以解决this的问题，省略bind(this)

// constructor(构造)钩子函数，可以定义state，如果不定义state，有没有钩子函数无区别
一但写了constructor钩子函数，就必须在此函数中写super(),此时组件才有自己的this，在组件的全局中都可以使用this关键字

setState更新是同步还是异步

1.setState会引起视图重绘
2.在可控的情况下是异步，非可控的情况下是同步
注意：
当改变数据想直接获取数据(同步)

react生命周期

随着我们对react理解和使用越来越多，生命周期的参考价值越来越高
函数列表：
componentWillMount:在组件渲染之前执行
componentDidMount:在组件渲染之后执行
showldComponentUpdate:返回true和false，true代表允许改变，false代表不允许改变
componentWillUpdate:数据在改变之前执行，(state,props)
componentDidUpdate:数据修改完执行(state,props)
componentWillReveiceProps:props发生改变执行
componentwillUnmount:组件卸载前执行

## 14.js内置对象
javascrpit 3种对象  自定义对象   内置对象   浏览器对象(js独有的，js API讲解)

内置对象：js语言自带的一些对象  
(1)日期对象 Date()   是一个构造函数，必须使用new 来创建日期对象

date对象是基于1970年1.1起的毫秒数(世界标准时间)

获取date总的毫秒数(时间戳)：  valueof()  getTime()  Date.now()  +new Date()

(2) 数组的内置对象

1. Array.isArray()  ／  instanceof    //检测是否是数组
2. push()   //在数组的末尾添加一个或者多个数组元素  给数组追加新的元素

注意：push() 完毕后，返回的结果是 新数组的长度

3. unshift() //在数组前面添加数组元素
4. pop() //删除数组最后的一个元素   无参数  返回值是删除的元素
5. shift() //删除数组的第一个元素  无参数  
6. sort()  //单位数可以，多位数有问题
7. indexOf()   //返回数组元素的索引值，不存在返回-1
8.toString() ／join(分隔符)    // 数组转换成字符串   

字符串操作
1. substr();  //字符串截取
2. concat();  //可以把多个参数添加到指定字符串的尾部
3. split(分隔符); // 字符串转换成数组
